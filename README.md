# SAGE Tutorial Edge App

This project is a SAGE edge application that captures an image using a camera and computes color statistics.

## Features

- Captures an image from a SAGE camera
- Computes mean RGB values
- Computes minimum RGB values
- Computes maximum RGB values
- Publishes results using the Waggle plugin API
- Uploads the captured image

## Files

- `main.py` - Main edge application
- `requirements.txt` - Python dependencies
- `Dockerfile` - Container build instructions
